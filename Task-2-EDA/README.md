# Task-2: Exploratory Data Analysis (EDA) & Business Intelligence

## Objective
Perform exploratory data analysis on a banking customer churn dataset to uncover patterns, trends, and relationships that explain customer churn behavior.

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Tableau Public
- GitHub

## Dataset
- Cleaned banking customer churn dataset from Task-1

## Key Analysis Performed
- Customer churn distribution
- Age distribution analysis
- Account balance distribution
- Correlation analysis between numerical features
- Business-focused visual insights

## Visualizations
- Count plots
- Histograms
- Box plots
- Correlation heatmap
- Tableau dashboard

## Deliverables
- EDA notebook with insights
- Tableau dashboard
- LinkedIn video explanation

## Outcome
The analysis highlights key churn drivers such as age group, balance patterns, and country-level churn differences, supporting data-driven decision-making.
